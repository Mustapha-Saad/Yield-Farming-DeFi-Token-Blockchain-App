# DeFi-Token-Blockchain-App

A Decentralised Finance(DeFi) Web App for staking, unstaking and transferring tether custom tokens to different nodes including a decentral bank, third party users and address wallets with an Airdprop functionality included.

## Main Technologies Used:

### Backend

1. [Solidity](https://docs.soliditylang.org/en/v0.8.12/)
2. [Mocha](https://mochajs.org/)
3. [Chai](https://www.chaijs.com/)

### Frontend

1. [ReactJs](https://reactjs.org/)
2. [Web3.js](https://web3js.readthedocs.io/en/v1.7.0/)
3. [Bootstrap](https://getbootstrap.com/)

### Development Environment

1. [Truffle With Ganache](https://trufflesuite.com/docs/index.html)

### Wallet Gateway

1. [Meta Mask](https://metamask.io/)
